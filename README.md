# ATM Management System (C++)

This is a simple ATM Management System developed using C++.

## Features
- PIN based login
- Check balance
- Deposit money
- Withdraw money
- Console based system
- GUI concept using wxWidgets

## Tools Used
- C++
- wxWidgets (concept)
- Windows

## How to Run
1. Open project in any C++ compiler
2. Compile and run `main.cpp`
